# Archived plotting routines (glacierware 2023)

These project-specific plotting scripts are **superseded** and kept only for
reference. Use the modern alternatives instead:

- **Standard comparison plots** — the built-in `glaciercore plot` CLI
  (see `../run_plot_example.sh`).
- **Power-map / general utilities** — `glaciercore_scripts/powermap_utils/`
  (visualizers, splitters, converters) shipped with glaciercore.
- **Result plots** (waterfall, dumbbell, lollipop, Notion DB) — the standalone
  scripts in the repo-root `scripts/` folder.

`project_plotting_utils.py` and `generate_plot_results_v1.py` target the old
pre-CLI workflow and the legacy power-map API.
